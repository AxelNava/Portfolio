project Compilador

Un compilador para un pequeño conjunto de instrucciones de C# escrito en C#, la última fase de desarrollo fue el código intermedio.

Mostrando proyecto...

Características principales

  • Implementación de analizador léxico propio
  • Uso del algoritmo LR(1) para el analizador sintáctico
  • Implementación propia del analizador semántico
  • Generación de implementación propia del código intermedio usando código de 3 direcciones

Este compilador puede generar el código intermedio dado el código fuente, devuelve los errores y evalúa operaciones matemáticas que puedan ser evaluadas usando notación prefija. Si el código fuente dado es válido, el programa crea un archivo de texto con el código intermedio.
project Aplicación de escritorio

Administración de usuarios para generar recibos de agua en PDF, manejo de usuarios con deudas automatizadas usando SQLite como base de datos.

Mostrando proyecto...

Características principales

  • Administración de usuarios
  • Subida de archivos Excel y csv
  • Incremento mensual de deuda de usuarios
  • Generación de recibos en PDF

Esta aplicación fue hecha como una propuesta para administrar los usuarios del servicio de agua, permite la carga de archivos Excel propios y CSV, genera el PDF con todos los recibos o solo un rango de usuarios e incremente mensualmente la deuda de cada uno de estos cuando la aplicación es abierta.
